# Memorias Tec
Memorias Tec es el proyecto final de la clase: Fundamentos del diseño interactivo. Éste consiste en una aplicación web que compila las memorias de alumnos y ex-alumnos sobre sus interacciones en la antigua biblioteca para destacar el legado emocional de la misma. 

## Memorias
Memorias Tec almacena testimonios en forma de texto, imágenes y audio. Cada testimonio tiene una o más etiquetas que permite agruparlos por sentimientos.
